Mysql基础知识

数据库(database)中 ，表(table)是一种结构化的文件,每个表有一个唯一的名字来标识。
表是由一个至多个列组成，每个列(column)有相应的类型。
表中的数据按行(row)排列，一行为表中的一个记录。

主键(primary key)是来标识行的唯一。主键可以没有但是进行操作可能会有出现麻烦。
主键是来标识行的唯一的所以主键所在的列在表中值不重复，可以没有主键但不能为NULL值
